Parameters:
_maxIterationsMaximal number of iterations. If not provided mbica::def::MAX_ITERATIONS will be set.
_epsilonParameter that says how fast iterations will converge. If not provided mbica::def::EPSILON will be set.
_guessMatrixInitial matrix that can be given to speed up the calculations. If not provided it will be set to random, orthonormal matrix.
_dWhDewhitening matrix. If not provided it will be calculated.
_whWhitening matrix. If not provided it will be calculated. Function that actually calculate FastICA.
XMatrix with data from which independent components will be calculated.
nICNumber of independent components to be found. If set to -1 it will take number of dimensions from data matrix.
muParameter mu. If not provided it's set to mbica::def::MU.
